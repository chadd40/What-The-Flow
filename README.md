# What The Flow

**What The Flow** is a minimalist, strategic Todo Application designed to keep your workflow clean, organized, and focused. This repository hosts all the official public releases, installers, and auto-update artifacts for the application.

## 🚀 Downloads & Installers

To download the latest version of **What The Flow**, head over to the [Releases Page](https://github.com/chadd40/WhatTheFlow-releases/releases/latest) and download the installer for your operating system:

- **macOS (Apple Silicon & Intel)**: Download the `.dmg` file.
- **Windows**: Download the `.exe` Setup file.

## 🔄 Auto-Updates
This app features seamless auto-updates! Once you have the app installed, it will automatically check this repository for new releases. 

When an update is published, the app will download it in the background and prompt you to install it, ensuring you are always running the latest version.
